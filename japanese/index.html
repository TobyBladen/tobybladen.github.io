プロジェクト

2018年

ロボキャット

修士のソフトウェア工学授業では、 子供向けにコンピューター・サイエンスの基礎を紹介する教育用のビデオゲームを開発する課題がありました。 UnityとC#、Blender、Inkscapeなどのツールを利用して6人のチームで3Dのビデオゲーム「ロボキャット」 を作ることに成功しました。私は主にGUIのプログラミングやロゴ・デザインの部分を担当しました。

レディー、セット、ジェット！

同じ修士のウェブ技術授業では、 クラスメートと2人で「レディー、セット、ジェット！」というフルスタックのアプリケーションを開発しました。 レディー、セット、ジェット！は、ユーザーの答えを分析して旅行先を勧めるクイズアプリです。 バックエンドはnode.js、express.js、EJS、SQLiteのデータベースなどを用い、 フロントエンドはBootstrapとjQueryにしました。 フロントエンドの部分ともバックエンドの部分とも両方担当してスキルを磨くいい機会だったと思います。

プラコト

Javaプログラミング授業の最終プロジェクトでは、 単独でボードゲーム「プラコト」のビデオゲーム版を開発することにしました。 あまり知られていない「プラコト」とは、私がよく趣味で遊んでいたバックギャモンの一種で、 ビデオゲーム化するのは楽しい挑戦でした。技術は主にJavaFXとラスターグラフィックエディタ「GIMP」を利用しました。

2017年

tobybladen.com

自分のホームページを作成することは、 コンピュータ・サイエンス専攻修士課程を行う前からも空いた時間で取り組んでいました。 自習したHTML、CSS、JavaScriptなどを活かしてJekyllで静的ウェブページを開発し、 GitHub Pagesにホストさせました。

2016年

今日のビデオ(BBC)

BBCウェブサイトの実験コーナー「今日のビデオ」 の日本語編集を担当していました。BBCは、ニュース動画の台本を自動翻訳し、 音声合成の技術で日本語の吹き替えを作るウェブアプリ「ALTO」を開発しました。 私が担当していた仕事は、日本の視聴者向けに短いニュースの動画を作成することによって、 初めてそのアプリを試験することでした。

「今日のビデオ」について詳しくはこちらをご覧ください

ザ・サロン(BBC)

ドキュメンタリーシリーズ「ザ・サロン」では、 日本語のニュース動画の編集を手伝っていました。日本語から英語へ翻訳し、 英語字幕の作成を補助しました。世界中に放送されたラジオドキュメンタリー番組の編集にも携わっていました。

「ザ・サロン」の他の動画はこちらをご覧ください。
