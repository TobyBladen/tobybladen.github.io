"

영어와 한국어를 하는 소프트웨어 개발자입니다. 2015년에 옥스퍼드 대학교의 일본어와 한국어 학과를 졸업하고, 영국의 공영 방송사인 BBC에서 경력을 쌓았습니다. 프로그래머로 전향하기 위해 2017년에 브리스톨 대학교에서 컴퓨터 과학 석사과정을 밟기로 해서 2019년 2월에 졸업했습니다. 개발 경험 있는 언어는 C, C++, C#, 자바, SQL, 자바스크립트 등이 있습니다.

프로젝트

2018년

로보캣

석사과정 소프트웨어 공학 수업에서 어린이를 위해 컴퓨터 공학 기초를 소개하는 교육적인 비디오게임을 개발하는 과제가 있었습니다. 유니티와 C#, 블렌더, 잉크스케이프 등을 이용해 6명 팀으로 "로보캣"("Robocat")이라는 3D비디오게임을 성공적으로 개발했습니다. 저는 주로 GUI 프로그래밍과 로고 디자인을 담당했습니다.

레디, 세트, 제트!

웹 기술 수업에서는 급우와 협력해 둘이서 "레디, 세트, 제트!"라는 full-stack 앱을 개발했습니다. "레디, 세트, 제트!"는 이용자 대답을 분석해 여행지를 추천하는 퀴즈 앱입니다. backend는 node.js, express.js, EJS, SQLite 데이터베이스 등, frontend는 부트스트랩과 jQuery로 작업했습니다. 백엔드와 프런트엔드를 둘 다 담당해서 실력을 향상시킬 수 있었던 즐거운 경험이었습니다.

플라코토

자바 프로그래밍 수업 최종 프로젝트는 "플라코토"라는 보드 게임의 비디오 게임버전 개발하기로 했습니다. 플라코토는 잘 알려져 있지 않은 게임인데, 백개먼의 일종으로 제가 항상 취미로 즐긴 보드게임입니다. 그래서 이를 비디오 게임화하는 것을 즐거운 도전이었습니다. 기술은 주로 자바FX와 화상 편집 소프트웨어인 GIMP를 이용했습니다.

2017년

tobybladen.com

자기 홈페이지를 작성하는 것은 컴퓨터 과학 석사과정에 입학하기 전에도 한가할 때 종사하고 있었습니다. 독학으로 공부하던 HTML, CSS, 자바스크립트 등 활용해 Jekyll로 정적 웹 페이지를 개발하고 호스트는 깃허브 페이지로 했습니다.

2016년

비디오 속 오늘 (BBC)

[영어 비디오] BBC의 실험적인 인터넷 뉴스 서비스 '비디오 속 오늘'의 일본어 편집을 담당했습니다. BBC는 뉴스 비디오의 대본을 자동 번역하고 음성을 입히는 기술로 일본어 더빙을 만드는 "ALTO"라는 웹 애플리케이션을 개발했습니다. 제가 담당한 일은 일본인 시청자를 대상으로 짧은 뉴스 비디오를 만들어서 그 애플리케이션을 처음으로 시험하는 일이었습니다.

더 살롱 (BBC)

[영어 비디오] 저는 "더 살롱"("The Salon")이라는 BBC의 다큐멘터리 시리즈에서 일본어 뉴스의 비디오 편집을 도왔습니다. 일본어를 영어로 번역하고 영어 자막 작성을 보조했습니다. 전 세계에 방송된 라디오 다큐멘터리 편집보조도 제 역할 중 하나였습니다.
