Projects

2018

Robocat

In the 'software engineering' module of my masters degree, we were asked to develop an educational video game to introduce school age children to fundamental Computer Science concepts. Working in an agile team of 6, we created a multilevel, full 3D game titled 'Robocat' using Unity/C#, alongisde software tools like Blender and Inkscape. I was primarily responsible for the UI programming in this project, alongside completing the logo design.

Ready, Set, Jet!

In the 'web technologies' unit of the same MSc course, I collaborated in a pair to build a full-stack quiz application, named 'Ready, Set, Jet' to help choose a travel destination. The application was built on a backend of node.js, express.js, and the EJS templating engine, encorporating an embedded SQLite database. The fully responsive app additionally employed Bootstrap and jQuery on the front end, and I was excited to develop my skills widely across the stack.

Plakoto

For the final solo project of my Java unit, I opted to develop a video game version of the board game Plakoto. Plakoto is a lesser-known, highly strategic version of backgammon I enjoyed playing as a hobby, so it was interesting challenge to adapt it into a video game. I primarily made use of JavaFX and the raster graphics editor GIMP.

2017

tobybladen.com

Developing my own personal website was undertaken in my spare time even before beginning my Computer Science degree. I used my self-taught HTML, CSS, and JavaScript skills to develop an original static site using Jekyll and hosted it using GitHub pages.

2016

Today in Video (BBC)

I was the Japanese language editor for experimental online news service ‘Today in Video’. A web app, ‘ALTO’, was developed by BBC News Labs to automatically translate video scripts and generate a voiceover in the target language using speech synthesis technology. I was responsible for trialling the software for the first time by creating short news packages for Japanese-speaking audiences.

Read more about the 'Today in Video' project in this BBC News article.

The Salon (BBC)

I assisted BBC Current Affairs in editing news video packages in Japanese as part of documentary series ‘The Salon’. I also translated from Japanese into English and aided the subtitling of the videos in English. Finally, I assisted the editing of a spin-off radio documentary that was broadcast worldwide.
